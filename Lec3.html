Nested Lists

Core Concept : A nested list is simply a list that's placed inside a list item (<li>) of another list. This is the key to creating outlines and hierarchical structures, allowing you to show relationships between different pieces of information. You can nest multiple levels deep to create complex, organized outlines, which is a very powerful tool.

Practical Use Cases : This technique is perfect for things like course outlines, where you have main topics with sub-topics nested underneath (e.g., DSA > Array, Linked List). The video also shows a detailed example of building a daily meal plan, using a mix of headings and multiple levels of ordered (<ol>) and unordered (<ul>) lists to structure the day.

Tables

Basics

Tables are used to display data in a grid of rows and columns. The fundamental tags are:

  • <table> : The main container for the whole table.
  • <tr> : Defines a Table Row.
  • <td> : Defines Table Data, which is a single cell inside a row.

To make the borders visible while you're building, you can add the border="1" attribute to the <table> tag.

Professional Structure(Semantic HTML)

This is the most important part for creating high-quality, professional tables.

  • <th> (Table Header) : You should always use this tag instead of <td> for the heading cells of your table (like "Name", "Age", "City"). It gives the headings semantic meaning, which is crucial for accessibility (screen readers) and search engines. Browsers also typically make <th> text bold and centered by default.
  • <thead>, <tbody>, <tfoot> : These tags group the different parts of your table.
    • <thead> wraps the row containing your table headers (<th>).
    • <tbody> wraps all the main data rows of the table.
    • <tfoot> wraps the footer row, which might contain totals or summaries.

Using these makes your code much cleaner, more readable, and easier for browsers and other developers to understand.

Advanced Layouts (Spanning Cells)

To create complex tables where cells are merged, you use two key attributes:

  • colspan (Column Span) : Adding colspan="2" to a cell tells it to stretch horizontally and take up the space of two columns.
  • rowspan (Row Span) : Adding rowspan="2" to a cell tells it to stretch vertically and take up the space of two rows.
